# Gather Round

The dinner dice that gets families talking.

This is a static first version of the Gather Round play experience. Families can roll the physical die, tap the matching side on the website, and receive a random prompt filtered by age range and Christian tone. The site also includes a digital die for families who want to play without the physical cube.

## Current MVP

- Six dice sides: Heart, Faith, Family, Friends, Tomorrow, Wild
- Age buckets: Preschool, Grade school, Middle school, High school
- Faith tone settings: Light, Clear, Devotional
- Random prompts with family-involvement helper text
- Physical-die-first flow with digital die fallback
- No accounts, login, or backend required

## Open Locally

Open `index.html` in a browser.

## Product Notes

The working physical concept is a 3-inch foam cube with five category faces and one Wild face. The Wild face uses the QR code as its icon so the die remains fully playable while still making the website easy to access at dinner.
